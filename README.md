# 塗料計算機 (PaintCalc)

塗料の硬化剤・希釈液の配合量を計算するAndroidアプリ

## 概要

高所作業現場で、塗料の配合をスマホで素早く計算するためのシンプルなツール。
スクロール不要の1画面UIで、手袋装着時でも操作しやすい設計。

## 機能

- 主液：硬化剤の比率入力（例: 10:1）
- 希釈率（%）入力
- 作成量（kg）入力
- 各成分の必要量を自動計算

## 開発環境

- Android Studio Ladybug (2024.2.x) 以降
- Kotlin
- Minimum SDK: API 26 (Android 8.0)
- Target SDK: API 35 (Android 15)

## パッケージ
```
com.nebukin.android.paintcalc
```

./gradlew assembleDebug
